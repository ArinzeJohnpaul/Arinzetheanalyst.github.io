sql analysis
on a financial platform

This project contains SQL queries designed to extract insights from a financial platform’s user activity. Key areas include identifying multi-product users, analyzing transaction frequency, calculating tenure and customer lifetime value (CLV), and converting currency data.

This SQL project explores key user behaviors on a financial platform to support strategic decision-making. It includes queries targeting user engagement, transaction patterns, activity recency, and estimated customer value.

The first query focuses on identifying highly engaged users who have at least one funded savings plan and one funded investment plan. These users are ranked by total confirmed deposits (converted from kobo to naira). This helps the business identify high-value, cross-product customers.

The second part segments users based on their monthly transaction frequency. Using common table expressions (CTEs), it calculates average transactions per user per month and categorizes them into high (≥10), medium (3–9), and low (3) frequency groups. This is useful for identifying power users and inactive accounts.

The third query identifies users who have transacted within the last 365 days. It classifies their accounts as either savings or investments using a CASE statement and measures inactivity by comparing the latest transaction date with the current date. This reveals which users are still active and in what capacity.

The fourth section estimates customer lifetime value (CLV) by combining transaction count and user tenure. It calculates average monthly transactions, projects them over a year, and multiplies by an estimated profit per transaction. This helps predict user value over time.

Several technical challenges were addressed in the project: handling divide-by-zero scenarios with NULLIF, maintaining currency precision with ROUND, segmenting complex logic using CTEs, and ensuring data integrity through careful joins and grouping.

The project is organized into four SQL files, each focusing on a different analytical goal. These insights support user segmentation, retention strategies, and value estimation..
