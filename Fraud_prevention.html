This project builds a machine learning model to detect fraud in public procurement. Using Python, we engineered key risk indicators like cost overrun and trained a Random Forest classifier. This project develops a machine learning system to detect fraudulent contracts in public procurement. Using Python, we generated a realistic dataset and engineered features like cost overrun percentage. We trained both an Isolation Forest and a Random Forest model to identify anomalies and classify fraud. The analysis revealed key risk indicators: high cost overruns, single-source awards, and low bidder numbers. The model serves as an effective early-warning system, enabling auditors to prioritize investigations, protect public funds, and strengthen governance by focusing on the highest-risk transactions.

The project is organized into four SQL files, each focusing on a different analytical goal. These insights support user segmentation, retention strategies, and value estimation..
